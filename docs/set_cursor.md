# Set Cursor

Set cursor location and rotation aligned to surface.

- **Description**: Raycast-based cursor placement tool. Supports edge and vertex snapping.
- **Controls**:
  - `LMB`: Place Cursor
  - `Scroll`: Cycle Edge Alignment
  - `S`: Toggle Snap (to vertex/edge/face center)
  - `RMB`: Done / Exit
  - `ESC`: Cancel
