# Interactive Hull

Fit a Convex Hull around marked faces.

- **Description**: Generates a convex hull mesh from marked faces and points.
- **Controls**:
  - `Space`: Create Hull
  - `LMB`: Mark Face
  - `A`: Toggle Point Mode (Add Points)
  - `C`: Toggle Coplanar Selection
  - `P`: Toggle Backface Rendering
  - `O`: Toggle Preview Culling
  - `D`: Toggle Depsgraph (Modifiers)
  - `Z`: Clear All
  - `1-7` / `Shift+Scroll`: Adjust Coplanar Angle
  - `RMB`: Done / Exit
  - `ESC`: Cancel

- **Point Mode Controls**: (Press `A` to enter)
  - `LMB`: Add Point
  - `S`: Toggle Snap
  - `C`: Toggle Limitation Plane
  - `Ctrl+Scroll`: Adjust Snap Threshold
  - `A`: Exit Point Mode
