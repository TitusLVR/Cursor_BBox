# Interactive Box

Fit a Bounding Box around marked faces or active object.

- **Description**: Places cursor and creates a bounding box with face marking support.
- **Controls**:
  - `Space`: Create Box
  - `LMB`: Mark Face
  - `Mouse Wheel`: Rotate Box
  - `A`: Toggle Point Mode (Add Points)
  - `C`: Toggle Coplanar Selection
  - `P`: Toggle Backface Rendering
  - `O`: Toggle Preview Culling
  - `D`: Toggle Depsgraph (Modifiers)
  - `Z`: Clear All
  - `1-7` / `Shift+Scroll`: Adjust Coplanar Angle
  - `RMB`: Done / Exit
  - `ESC`: Cancel

- **Point Mode Controls**: (Press `A` to enter)
  - `LMB`: Add Point
  - `S`: Toggle Snap
  - `C`: Toggle Limitation Plane
  - `Ctrl+Scroll`: Adjust Snap Threshold
  - `A`: Exit Point Mode
